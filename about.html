About This Portfolio

This website is a personal portfolio created by Sandeep Hamal as part of the Web Technology assessment for the MSc. Computer Science (Conversion) course at the University of Law.

The portfolio is designed using HTML, CSS, and JavaScript. It demonstrates fundamental web development concepts such as page navigation, layout design, form handling, and simple applications.

Inside this portfolio, you will find different sections including Home, Skills, Projects, and Contact. The Projects section includes beginner-level applications such as a Weekly Consolidate Task layout and a simple Inventory Management System.

The main purpose of this portfolio is to showcase my learning progress in web development and to present my work in a clean, structured, and user-friendly way.
